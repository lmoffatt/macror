>>help likelihood

likelihood patch-alias experiment-alias
      Computes the likelihood of the provided parameters of the specified
      patch model for the given experiment. The results are stored in myLikelihood.


likelihood patch-alias experiment-alias [option-alias] [likelihood-newp-alias]
      Computes the likelihood of the provided parameters of the specified
      patch model for the given experiment using the specified options and
      storing the results in the provided new alias.
